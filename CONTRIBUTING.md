# Contributing

We welcome contributions to improve this quickstart.

## How to Contribute

1. **Open an issue** — describe the improvement or bug
2. **Fork the repo** — make changes on a feature branch
3. **Run `terraform validate`** — ensure all modules validate cleanly
4. **Submit a pull request** — reference the issue

## Guidelines

- All Terraform must pass `terraform validate`
- Follow HashiCorp naming conventions
- Agent configs must include guardrails and instructions
- Include variable descriptions and type constraints
- All contributions must be compatible with the Apache 2.0 license

## Questions?

Open an issue or reach out at [proximaintel.com/contact](https://proximaintel.com/contact).
